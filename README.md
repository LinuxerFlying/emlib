# emlib

*Emlib is for embedded lib, Enjoy it!*

## Emlib include modules

### OS:
1. [*OK*]Abstract lock mechanism.
2. Thread Task manage.
3. I/O
4. Socket
5. DBMS 
6. Memory Manage 

### Data Struct 
1. Double Linked list.
2. Queue
3. [*OK*]String
4. Stack
5. ...

### Algorithm
1. Sort
2. Search
3. Graph
4. Tree
5. ...

### Tools:
1. [*OK*]Base log manage.


## How to Build emlib
1. Download emlib
2. cd emlib
3. cmake .
4. make 
5. ./emlib(Run all emlib test cases.)
